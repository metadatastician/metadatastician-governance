# Repository instructions

This repository is The Metadatastician's organisation-wide adopting governance profile. It particularises and adopts pinned upstream Hyperpolymath standards; it must not silently redefine or duplicate them.

Read `0-AI-MANIFEST.a2ml`, `README.adoc`, `ESTATE-PROFILE.adoc`, `AUTHORITY-MAP.adoc`, and `ADOPTION-AND-DEVIATIONS.adoc` before editing. Canonical local policy is in the root governance documents; machine-readable descriptiles mirror declared state and do not outrank them. Project files in `templates/` and `.machine_readable/templates/` are templates, not adopted bindings.

Do not edit generated files directly; none are currently declared. Do not change licences, upstream coined names, MAA, or evidence labels as a side effect. Proposals are not authorised policy until `CHANGE-PROCEDURE.adoc` is completed. Validate A2ML with the canonical validator when available, check links and `git diff --check`, and record tool/version and failures.
